# Event Pro

## Project Description

Event Pro is a web-based tool that helps users organize and manage events such as parties, conferences, or workshops. It allows users to input event details, manage attendees, assign tasks, visualize progress, and analyze event data to improve planning efficiency.

## Features

### 1. Event Details and Attendee Management
- Add event details (date, time, location, description)
- Manage attendees (name, contact info, RSVP status)

### 2. Task Assignment and Timeline
- Assign tasks to attendees or self
- Set deadlines and update task status
- Visual timeline for tracking event preparation

### 3. Data Visualization and Event Analysis
- Charts showing attendee breakdown (e.g., by role or preferences)
- Task progress tracking
- Insights to identify areas for improvement

## Technologies

- Python
- Flask (or Streamlit, TBD)
- SQLite or PostgreSQL
- Pandas / Plotly / Seaborn for data visualization
- GitHub for version control



## Getting Started

1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `python app.py`
